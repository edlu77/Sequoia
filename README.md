# seQuoia

[Live link](https://sequoiaapp.herokuapp.com/)

seQuoia is a Quora-inspired web app that allows users to ask and receive answers to questions. Questions are organized by topic.

# Technologies

* Ruby
* Rails
* Javascript
* React/Redux
* HTML

# Feature Highlights

* Main Feed

The main feed contains a randomized list of questions and answers on the website. Users may click on a topic on the sidebar in order to filter the feed to only show questions and answers belonging to that topic.

* Question Show Page

Each question

* Responsive Search

The search feature updates results on each key press and displays them in a list. Users may then click a question to view that question's show page.
